## 🎯 О нас

Мы создаем технологии для выживания и роста малого бизнеса. 
Наша миссия — предотвратить закрытие МСП из-за финансовой неграмотности и отсутствия своевременной аналитики.

### 📊 Проблема
- В 2024-2025 гг. в России закрывается в **1.5 раза больше** компаний, чем открывается
- В Пермском крае под угрозой **107,000 субъектов МСП** и **518,000 рабочих мест**
- Основная причина — предприниматели не видят финансовые проблемы вовремя

### 💡 Решение
**НеоФин.Документы** — AI-платформа для анализа финансовой отчетности и управления рисками

## 🚀 Ключевые возможности MVP

### НеоФин.Документы

✅ **Автоматический анализ финансовой отчетности**
- Поддержка МСФО и РСБУ
- Извлечение 15+ финансовых показателей
- Dual Leverage расчет (total & debt-only)

✅ **Truthful Scoring**
- Оценка достоверности данных
- Confidence levels для каждого показателя
- Auto-detect профиля компании (generic / retail_demo)

✅ **AI Runtime**
- Выбор AI-провайдера (Ollama, внешние API)
- Локальный запуск для безопасности данных
- Fallback механизмы

✅ **Benchmark'и**
- Отраслевые сравнения (retail_demo)
- Periodic scoring (reported / annualized_q1 / annualized_h1)
- Peer context анализ

## 📈 Экономический эффект

**При внедрении в Пермском крае:**
- 💼 Сохранение **15,000+ рабочих мест** (при спасении 5% МСП)
- 💰 **+2-3 млрд руб.** в региональный бюджет ежегодно
- 📊 Снижение риска ошибок в отчетности на **85%**
- ⚡ Ускорение проверки кредитных заявок с 5-7 до 1-2 дней
